# Data-Visualization-Demo
[This site](http://imamathwiz.github.io/Data-Visualization-Demo/) visualizes activity data using the D3 library to display a line chart and pie chart of the data.

## Libraries Used
* Backbone
* D3
* Lo-Dash
* jQuery
